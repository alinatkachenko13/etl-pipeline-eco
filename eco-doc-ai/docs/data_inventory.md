# Инвентаризация данных

## Общая характеристика данных
Проект работает преимущественно с неструктурированными и полуструктурированными текстовыми данными в области экологической и природоохранной документации.

## Источники данных

### 1. Внутренние документы компании
- архив разработанной экологической и природоохранной документации
- готовые проектные документы
- рабочие материалы специалистов

### 2. Шаблоны и типовые разделы
- шаблоны проектной документации
- типовые блоки и разделы
- стандартные формулировки и структуры документов

### 3. Методические материалы
- внутренние инструкции
- методические рекомендации
- поясняющие документы специалистов

### 4. Исторические проекты
- примеры завершённых документов
- ранее подготовленные проекты
- примеры решений по аналогичным задачам

### 5. Нормативно-правовая база
- федеральные законы
- подзаконные акты
- регламенты профильных ведомств
- публичные нормативные документы и справочники

### 6. Данные для обучения и тестирования
- подготовленные текстовые корпуса
- размеченные фрагменты документов
- наборы вопросов и ответов по экологической документации

## Форматы файлов
- DOC
- DOCX
- PDF
- TXT
- XLS
- XLSX
- JPG
- JPEG
- PNG
- HEIC

## Особенности данных
- документы разнородны по структуре
- часть документов содержит таблицы
- часть материалов представлена в виде сканов
- возможны фотографии документов с телефона
- качество изображений может быть низким
- в документах могут встречаться типовые разделы и повторяющиеся формулировки

## Основные проблемы данных
- неструктурированный текст
- смешение текста, таблиц и изображений
- плохое качество сканов
- разный формат оформления документов
- ограниченный объём размеченных данных
- ограниченный доступ к части внутренних материалов

## Использование данных в проекте
Данные будут использоваться для:
- извлечения текста
- OCR
- анализа структуры документов
- формирования базы знаний
- retrieval
- question-answering
- суммаризации
- извлечения требований
- тестирования качества решения
