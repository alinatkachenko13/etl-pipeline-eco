# Eco Doc AI

## Цель проекта

**Eco Doc AI** — AI-сервис для анализа экологической и природоохранной документации. Сервис помогает извлекать требования из текстов, формировать краткие саммари, отвечать на вопросы по содержанию документов и генерировать черновики фрагментов документации.

## Целевой пользователь

Экологи-проектировщики и специалисты, работающие с нормативной и проектной экологической документацией.

## Входные форматы данных

- **Документы:** PDF, DOCX, TXT
- **Таблицы:** XLS, XLSX
- **Изображения (сканы, фото):** JPG, JPEG, PNG, HEIC

## Ожидаемый результат

Структурированный анализ загруженных материалов: извлечённые требования и факты, ответы на вопросы пользователя, саммари и черновики текстовых фрагментов на основе исходных данных.

## MVP первой версии

- Загрузка документа (поддерживаемые форматы)
- Извлечение текста из файлов
- Поиск информации по содержимому
- Question-answering по документу
- Краткое саммари по загруженным материалам
